# 05 - Landing Page Workshop

## เป้าหมายบทเรียน

ให้นักเรียนรวมสิ่งที่เรียนจากบท 1-4 มาสร้าง landing page ที่มี hero, feature section และ pricing section

## Section ในตัวอย่าง

- Sticky navbar
- Hero section
- Feature cards
- Pricing card

## จุดที่ควรชี้ในไฟล์ HTML

- `sticky-top` ทำให้ navbar ติดด้านบน
- Hero ใช้ `row align-items-center g-5`
- Media placeholder ใช้ `ratio ratio-16x9`
- Feature cards ใช้ `row-cols-1 row-cols-md-3`
- Pricing ใช้ `card shadow-sm`

## Live Coding Flow

1. เปลี่ยนข้อความ brand และ headline
2. เพิ่มปุ่ม secondary ใน hero
3. เพิ่ม feature card ใบที่ 4 แล้วดู responsive behavior
4. เปลี่ยน pricing card ให้มีรายการ feature ด้วย `<ul>`
5. เพิ่ม footer ด้วย `bg-dark text-white`

## แบบฝึกหัด

- ทำ landing page สำหรับคอร์สเรียน 1 หน้า
- เพิ่ม testimonial section
- เพิ่ม FAQ section ด้วย accordion จากบท JavaScript

## คำถามท้ายบท

- Landing page ควรมีกี่ section ขั้นต่ำ
- Hero section ต้องสื่ออะไร
- ทำไมต้องคิด mobile layout ตั้งแต่แรก
