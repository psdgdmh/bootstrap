Bootstrap Essentials

สอนการใช้ container, typography, spacing, color และ utility classes พื้นฐาน

Typography

ใช้ `display-*`, `lead`, `fw-*`, `text-*` เพื่อจัดลำดับความสำคัญของข้อความ

Spacing

ใช้ `p-*`, `m-*`, `gap-*` เพื่อลดการเขียน CSS เองในงาน layout พื้นฐาน

Colors

ใช้ `text-bg-*`, `bg-*`, `border-*` เพื่อสร้าง visual state อย่างรวดเร็ว
